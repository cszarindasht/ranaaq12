وب نرم‌افزارها

نسخه‌های وب‌بیس برای دسترسی آسان

در این بخش، جایگزین‌های آنلاین و وب‌بیس برای نرم‌افزارهای اصلی رشته‌های کامپیوتر معرفی شده است. بدون نیاز به نصب، مستقیماً در مرورگر کار کنید.

نرم‌افزارهای مورد نیاز و منابع آموزشی (نسخه‌های وب‌بیس)

در ادامه، برای هر نرم‌افزار لیست‌شده، نسخه وب‌بیس یا جایگزین آنلاین مناسب را معرفی می‌کنم. این گزینه‌ها اجازه می‌دهند بدون نصب روی سیستم محلی، مستقیماً از طریق مرورگر وب (مانند Chrome یا Edge) استفاده کنید. لینک‌های مستقیم به صفحات مربوطه اضافه شده و در صورت نیاز، به حساب کاربری رایگان (مانند Adobe ID یا Microsoft Account) نیاز دارید. اگر گزینه‌ای مستقیماً موجود نباشد، بهترین جایگزین آنلاین را پیشنهاد می‌دهم. هدف این است که دانش‌آموزان بدون سیستم بتوانند از طریق گوشی وارد شوند و استفاده کنند.

نرم‌افزارهای اصلی:

Windows 10/11:

از Windows 365 Cloud PC یا Azure Virtual Desktop استفاده کنید. این سرویس‌ها ویندوز کامل را در مرورگر شبیه‌سازی می‌کنند.

Windows App for Azure Virtual Desktop

Microsoft 365 for the web (شامل Word، Excel، PowerPoint و OneNote).

office.com

photoshop.adobe.com

illustrator.adobe.com

vscode.dev

Adobe Dreamweaver CC 2025:

xd.adobe.com

Replit یا Glitch (محیط توسعه وب آنلاین با PHP/MySQL بدون سرور محلی).

CodeSandbox یا Heroku Free Tier (سرور آنلاین برای تست PHP/Apache/MySQL).

codesandbox.io (برای پروژه‌های وب‌بیس)

WordPress 6.6:

WordPress.com (میزبانی و ویرایش کامل سایت بدون نصب).

wordpress.com

Adobe Premiere Pro 2025:

Adobe Express Video Editor (ویرایش ویدیو آنلاین با ابزارهای پایه).

express.adobe.com/video/editor

Adobe Audition 2025:

Adobe Podcast (ویرایش و ضبط صدا با AI در مرورگر).

podcast.adobe.com

Adobe Captivate 2025:

Articulate Rise 360 (ساخت دوره‌های آموزشی آنلاین بدون نصب؛ جایگزین قدرتمند). نسخه آزمایشی رایگان.

rise.articulate.com

Adobe Photoshop Elements 2025:

Adobe Elements Web App (Beta) (ویرایش عکس ساده در وب).

elements.adobe.com
